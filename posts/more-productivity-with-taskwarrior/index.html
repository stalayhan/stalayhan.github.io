More Productivity With Taskwarrior

Introduction to Taskwarrior

I’ve actually been trying to use taskwarrior for my primary task managemnt tool about it for months without any progress.

But today is the day. I am aware of the taskwarrior power when you configure optimal way for your requirements. I don’t explain why you have to choose taskwarrior in this post. If you are wondering how it is awesome, you can visit the official site for great explanation.

TLDR; I put here my mostly used taskwarrior commands and configurations.

How to install taskwarrior

apt install task			# for debian users

Most simple taskwarrior commands

Basic three commands; add, list, done

task add "Buy a new CBR 600RR"
task list
task <ID> done

Add task with priority

task add priority:H laylaylom		# laylaylom is always HIGH priority task

See the next high priority task, type next; by the way this command is default, just type task is enough.

task next				# or task

Permanently delete a task

task <ID> delete

.taskrc

task show

I add below settings to the end of ~/.taskrc file to see public holidays in Turkey.

calendar.holidays=full
color=on
include /usr/share/task/holidays.tr-TR.rc

Command line syntax

This section just includes a few images to show simple usage. You can read further details about it at [2].

Need to know most important thing is that the all parts to the syntax (filter, command, modifications, miscellaneous), each part is optional.

Modification example;

Best practices with all commands

task <ID> modify project:<project_name>				# Assign a project to your tasks if possible
task <ID> modify due:<date>					# Assign due dates where appropriate
task <ID> modify priority:<priority_level>			# If you know the priority of a task
task <ID> modify +<tag> +<more_tag>				# Add useful tags to a task
task <ID> modify +next						# Add the special tag +next to a task, to boost its urgency
task <ID> modify depends:<OTHER_ID>				# Some tasks may depend on another task
task <ID> start							# When you start working on a task, mark it.

task config urgency.user.tag.problem.coefficient 4.5
task config urgency.user.project.Home.coefficient 2.9		# boost the whole project

task entry.after:today-4days list				# Show tasks I added in the last 4 days.
task entry:yesterday list					# Show tasks I added yesterday.
task entry.after:now-1hour list					# Show tasks I added in the last hour
task end.after:2015-05-01 and end.before:2017-05-31 completed	# Show tasks I completed between a date range.
task end.after:today-1wk comoleted				# Show tasks I completed in the last week
task project:This or project:That list				# Show tasks in This project or That project
task project:This and \( priority:H or priority:M \) list	# More complex algebric filters
task project: list						# Search for tasks with no project

task rc.report.next.sort=due-,urgency- next			# Temp changing the sorting of a report.

task project:OLDNAME modify project:NEWNAME			# Moving all tasks to a new project.
task project:OLDNAME and status:pending modify project:NEWNAME	# Moving all pending tasks to a new project.

task add project:Home.Kitchen Clean the floor			# Using a project hierarchy
task add project:Home.Kitchen Replace broken light bulb
task add project:Home.Garden Plant the bulbs
task project:Home.Kitchen count
task project:Home.Garden count
task project:Home count

task +home list							# List tasks that have a specific tag.
task -home list							# List tasks that do not have a specific tag.
task tags.any: list						# List tasks that have any tags.
task +TAGGED list
task tags.none: list						# List tasks that have no tags.
task +this and +that list					# List tasks that have two specific tags.
task +this xor +that list					# List tasks that have just one of two specific tags.
task tags							# What tags am I currently using ?

task due.after:yesterday and due.before:tomorrow list		# List tasks due today.
task +DUE -DUETODAY list					# List tasks that are due, but not today.
task +WEEK list							# List tasks that are due this week.
task +OVERDUE list						# List task that are overdue.
task <ID> info							# What virtual tags are present for this task ?

task add "Doe the thing" due:2018-07-25T18:00 recur:weekly	# I want to make a task that is due at 18:00 every Wednesday, 
								# starting this coming Wednesday.
task add "Buy a new book" due:28th recur:monthly until:now+1yr	# I want a reminder to buy a new book every month, but only for a year.

task config -- uda.priority.values H,M,,L			# Make priority L sort lower than no priority
task config -- uda.priority.values OMG,DoIt,LaLa,Nope		# I need more priority values for my workflow.
task 1 modify priority:						# How do I remove the priority from a task ?

task color
task color legend
task rc._forcecolor:on rc.defaultwidth:120 rc.detection:off	# When task output goes to a file or pipe, all color is lost. Force it.

task _get 12.description					# Get just the description for task 12.
task _get 12.entry 12.modification				# Show the creation timestamp, and last modification date for task 12.
task _get context.width context.height				# Get the dimensions of my terminal window.
task add "Buy a new book" due:oem wait:due-4days		# Add a task, and set the wait date to 4 days before the due date.
task add "Buy a new mouse" due:12.due				# Add a task, and use the same due date as task 12.
task _get 12.due.week						# Get the week number on which task 12 due.
task _get rc.report.next.columns				# Get the columns used in the next report.
task show report.next.columns
