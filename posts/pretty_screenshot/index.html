Take Pretty Screenshot

As you may imagine after complete a project you need to create a pretty, clear, and glaring README file. I believe it’s a de facto standard for open-source projects anymore. If you want to create an imposing markdown, you should use visual materials and efficient tongue for your audience.

Here is one of my favorite tool for creating awesome README and editing screen shots.

The App is Shutter

Unfortunately this app does not exist in the official Manjaro Repos. (I use Manjaro for my daily use by the way).

Install using with AUR if you are a Manjaro/Arch user.

yay -S shutter

Another way to install using with Snap

snap install shutter

Also there is an official options in the Upstream URL for Debian/Ubuntu.
