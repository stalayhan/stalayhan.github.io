LF Mentorship Session

Notes from LF Live Mentorship Session: Best Practices to Getting Your Patches Accepted

Important notes and links:

Start watching mailing lists of your interest,

  • https://lore.kernel.org/lists.html

Take LFD103 which is a free LF training that offers tips on how to start contributing. Check this site to see if you can fix any bugs

  • https://syzkaller.appspot.com/upstream
  • https://training.linuxfoundation.org/training/a-beginners-guide-to-linux-kernel-development-lfd103/
  • https://kernelnewbies.org/ is a good start

also check the #kernenewbies channel on OFTC

Checkout Patchwork is where kernel subsystem patches can be found.

  • https://patchwork.kernel.org/

Please find the presentation in the link below:

  • https://git.sr.ht/~gregkh
